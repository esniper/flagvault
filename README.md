# flagvault.com

Company holding page for FlagVault Technologies. Built with Next.js (App Router), Tailwind CSS, and TypeScript. Deployed on Vercel.

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
```

## Deploy

Push to the connected Vercel project. No environment variables or custom config required.

## Structure

- `/` — Home page with discontinued product notice
- `/privacy` — Privacy policy
- `/terms` — Terms of service
- `/support` — Support contact

All site copy is centralised in `lib/config.ts`.
